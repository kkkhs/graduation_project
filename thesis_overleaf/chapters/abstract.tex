\begin{abstract}
  遥感场景下的微小船舶检测在海上监管、港口监测、海事执法和应急救援等领域具有重要应用价值。然而，中分辨率遥感图像中船舶目标尺度小、纹理弱，且易受海浪、云层、岸线和港口设施等复杂背景干扰，导致检测任务存在漏检率高、误检率高和模型泛化能力不足等问题。针对上述问题，本文围绕公开数据集 LEVIR-Ship，开展了微小船舶检测算法对比实验与系统实现研究。

  本文首先对遥感船舶检测、小目标检测和通用目标检测相关研究进行了梳理，在统一数据划分和评测口径的基础上，选取 DRENet、FCOS 和 YOLO26 三类具有代表性的检测模型开展对比实验。实验采用 AP50、AP50-95、Precision、Recall 和 F1 作为主要评价指标，并结合可视化结果对模型差异进行分析。当前已完成的实验结果表明，DRENet 在 LEVIR-Ship 数据集上的 AP50 为 0.7949，YOLO26 的 AP50 为 0.7950，FCOS 的 AP50 为 0.7700。DRENet 在复杂背景与微小目标表征方面具有一定优势，YOLO26 在整体精度与工程部署便利性之间表现较为均衡，FCOS 则提供了 anchor-free 路线下的稳定对比基线。

  在系统实现方面，本文将算法能力封装为一套面向本科毕业设计场景的轻量级检测系统，采用 React 构建前端界面，采用 FastAPI 构建后端服务，并通过 SQLite 实现任务记录与结果持久化。系统支持单图与批量任务提交、模型切换、进度查询、检测结果可视化和历史结果回看，打通了从模型推理到结果展示的完整链路。实践表明，该系统能够满足毕业设计阶段“可运行、可演示、可追溯”的基本要求。

  本文的工作说明，在统一评测协议下开展多模型对比，并结合系统化实现与结果分析，能够更全面地评估不同检测范式在遥感微小船舶任务中的适用性。相关研究结果和系统实现可为后续的模型优化、消融实验和工程部署提供基础。
\end{abstract}

\begin{abstract*}
  \noindent %西文首段不缩进
  Tiny ship detection in remote sensing images is of great significance for maritime supervision, port monitoring, law enforcement and emergency response. However, ship targets in mid-resolution remote sensing images are usually very small and easily affected by complex backgrounds such as waves, clouds, coastlines and harbor facilities, which makes accurate detection difficult. To address this problem, this thesis conducts comparative experiments and system implementation for tiny ship detection based on the public LEVIR-Ship dataset.

  First, related studies on remote sensing ship detection, small object detection and general object detection are reviewed. Then, under a unified data split and evaluation protocol, three representative models, namely DRENet, FCOS and YOLO26, are selected for comparative experiments. AP50, AP50-95, Precision, Recall and F1 are adopted as the main evaluation metrics, together with qualitative analysis based on visualization results. The completed experimental results show that DRENet achieves an AP50 of 0.7949 on LEVIR-Ship, YOLO26 achieves 0.7950, and FCOS achieves 0.7700. DRENet shows advantages in representing tiny targets under complex backgrounds, while YOLO26 provides a balanced performance in terms of accuracy and engineering deployment.

  In addition, a lightweight detection system is implemented for the bachelor thesis scenario. The frontend is built with React, the backend is built with FastAPI, and SQLite is used for task persistence and result storage. The system supports single-image and batch inference, model switching, task progress query, visualization of detection results, and history review, thereby forming a complete workflow from model inference to result presentation.

  The study indicates that, under a unified evaluation protocol, combining multi-model comparison with system implementation can provide a more comprehensive understanding of the applicability of different detection paradigms to tiny ship detection in remote sensing scenes. The results of this thesis can serve as a basis for subsequent optimization, ablation studies and practical deployment.
\end{abstract*}
