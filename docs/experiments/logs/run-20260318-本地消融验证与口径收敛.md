### 4.3 DRENet 非 512 尺寸失败验证

命令：

```bash
source .venv/bin/activate && PYTHONPATH=. python tools/run_predict.py \
  --config configs/models.yaml \
  --image experiment_assets/datasets/LEVIR-Ship/test/images/GF6_WFV_E131.2_N35.8_20200910_L1A1120034678-3_1536_16384.png \
  --mode single \
  --model drenet \
  --conf 0.25 \
  --iou 0.50 \
  --imgsz 640
```

首个真实报错：

```text
RuntimeError: The size of tensor a (400) must match the size of tensor b (256) at non-singleton dimension 1
```

定位：

- 报错来自 `experiments/drenet/DRENet/models/common.py`
- 当前本地 DRENet 插件在 `imgsz != 512` 时会触发内部 shape mismatch

结论：

- DRENet 当前不能稳定支持任意推理尺寸覆写
- 因此不适合纳入“统一输入尺寸敏感性主表”
- 但 DRENet 仍可完整参与“三模型统一阈值消融”
