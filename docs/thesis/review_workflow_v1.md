## 每章审校检查项
- 结构完整：背景/问题/方法或分析/小结
- 术语统一：与项目文档一致
- 引用规范：占位 key 合法且无重复冲突
- 图表规范：标签命名符合 `fig:chX_xxx` / `tab:chX_xxx`
- 数据一致：不与实验记录和系统文档冲突
